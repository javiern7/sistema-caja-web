# Sistema Caja Web

Base técnica frontend del proyecto usando React, Vite, TypeScript, React Router, TanStack Query, Zustand, React Hook Form, Zod y Tailwind CSS.

## Estructura

- `src/layouts`: layouts de la aplicación
- `src/routes`: rutas y guards
- `src/modules`: módulos funcionales
- `src/components`: componentes compartidos
- `src/services`: cliente API y servicios
- `src/store`: estado compartido
- `src/styles`: estilos globales
- `docs-tecnicos`: notas técnicas específicas del frontend
